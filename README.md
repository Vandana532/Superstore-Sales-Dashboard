# Superstore Sales Dashboard

## Project Overview

This project focuses on analyzing sales performance, profitability, and product-wise revenue trends using Superstore sales data. The goal was to build an interactive dashboard for better business decision-making.

## Tools Used

* Microsoft Excel
* Power BI
* DAX
* Pivot Tables
* Data Cleaning

## Key Dashboard Features

* Product-wise Sales Analysis
* Monthly Sales Trends
* Yearly Sales Growth
* Profit vs Gross Sales Comparison
* KPI Reporting
* Interactive Filters and Visualizations

## Key Insights

* Certain products contributed significantly higher revenue
* Monthly sales patterns identified peak-performing periods
* Sales showed year-over-year growth
* Higher units sold did not always result in higher profit

## Business Impact

This dashboard helps businesses understand product performance, profitability trends, and sales growth opportunities.
